## 1-A 戰鬥手感優化

### 珠子移動與消除節奏

| 問題 | 現狀 | 優化方案 |
|------|------|----------|
| 消除動畫僅 300ms 靜態清除 | `ANIM_CLEAR = 300` 無視覺衝擊 | 加入「爆散粒子」效果，珠子消除時向外噴出 8 個小粒子，持續 250ms |
| 下落補充無緩動 | `ANIM_DROP = 200` 線性 | 改為 ease-out cubic，上方珠子落下有重量感（先快後慢彈跳） |
| 移動時無視覺預覽 | 拖拉後直接交換 | 半透明預覽：移動中的珠子顯示 70% 透明度，目標位置顯示虛線框 |
| COMBO 文字無動態 | 靜態文字 500ms | COMBO 數字從 0.5x 縮放彈出至 1.2x 再縮回 1.0x，每 COMBO +1 文字顏色升級（白→黃→橘→紅→彩虹） |

### 傷害數字演出

- 普通傷害：白色數字，向上飄移 +40px，淡出，持續 800ms
- 屬性剋制（1.5x）：橘色數字，字體加大 20%，附帶火花粒子
- 大 COMBO（5+）：金色數字，字體加大 40%，附帶光暈
- 治療數字：綠色 +數值，向上飄移
- MISS：灰色 MISS 文字

### 攻擊演出節奏改善

現有 `ANIM_ATTACK = 400ms`，建議改為分段：

1. **蓄力**（100ms）：攻擊角色圖示輕微縮小（scale 0.9）
2. **出擊**（150ms）：圖示快速放大（scale 1.15）＋向敵方位置移動閃光線條
3. **命中**（150ms）：敵方圖示震動（CSS shake），傷害數字浮現
4. **回彈**（100ms）：恢復原位，敵方 HP 條動態減少

### Boss 受擊演出

- HP 每減少 25%：Boss 圖示閃紅 + 螢幕輕微震動（CSS `translate` ±3px，3 次往返）
- 擊破波次：白色閃光覆蓋全螢幕 100ms，敵方圖示碎裂消失動畫

### 技能釋放演出

- 特殊技釋放前：0.5 秒全畫面暗化（overlay opacity 0.6），角色名稱顯示

- 技能效果粒子：根據屬性不同（火焰/水流/樹葉/光芒/暗影）

```

| 受擊 | 隊伍受到攻擊 | 衝擊聲 |
| 治療 | RCV 觸發 | 清脆「叮」聲 + 輕柔氣泡聲 |

| 升級 | 角色等級提升 | 上升音階 + 小鈴聲 |
| 親密升級 | 親密度突破 | 溫柔鐘聲 + 心跳聲 |

  // BGM 淡入淡出切換（500ms cross-fade）

# Part 2：劇情內容規劃

## 2-A 世界觀設定文件

### 世界背景：「雙界裂縫」

遠古時代，這個世界原本只有一面。
人類文明在這片名為「陽界」的土地上繁榮發展，直到三百年前，一場被稱為「大崩」的災難撕裂了世界的構造——陽界的地下深處，一個鏡像的空間「幽界」開始與陽界重疊。

幽界並非只是黑暗的倒影。它有自己的生態、自己的規則，以及最危險的居民——「異獸」。

異獸並非普通的動物，而是幽界能量結晶化的產物。它們通過「裂縫」滲透進陽界，侵蝕土地、腐化生命。裂縫的分布不均，某些地區幾乎已完全被幽界吞噬，那些地方被稱為「侵蝕地帶」。

人類對此並非束手無策。一部分人類，在大崩之後開始覺醒特殊能力——他們的感官超越了正常人類的極限，能夠感知、操縱、甚至與幽界能量直接互動。這些人，被稱為「超感者」。

### 超感者的三種類型

**哨兵（Sentinel）**

哨兵是超感者中攻擊性最強的一類。他們的五感超越人類極限數十倍：能在三公里外聽見耳語，在完全黑暗中看清毫髮，嗅出異獸的氣息如同旁邊的炭火。

然而，超感的代價是沉重的。哨兵長期處於「感官過載」的邊緣——過強的感知會讓他們痛苦、失控，嚴重時甚至會陷入昏迷或暴走。因此，哨兵需要「嚮導」的協助才能穩定運作。

哨兵的戰鬥風格偏向近身爆發，能將超感能量凝結為屬性攻擊。

**嚮導（Guide）**

嚮導是哨兵的天然搭檔。他們無法直接感知幽界，但具備一種特殊能力：「共鳴錨定」——能夠向哨兵發出穩定的能量頻率，讓哨兵的感官過載得到壓制，維持戰鬥狀態。

沒有嚮導的哨兵，如同沒有繩索的登山者。

嚮導的戰鬥能力相對較弱，但他們能強化隊友、治癒傷勢、以及執行「共鳴技能」——利用與哨兵的精神連結釋放遠超個人極限的合體技。

**越域者（Crosser）**

越域者是最稀少，也是最神秘的超感者類型。他們能夠自由穿越陽界與幽界之間的裂縫，在兩個世界之間自如移動。

越域者往往同時具備哨兵與嚮導的部分能力，但都不如專精者強大。他們最大的價值在於「探索」——深入其他人無法抵達的幽界深層，獲取情報或執行特殊任務。

越域者的存在非常罕見，在整個世界已知的超感者中，越域者不超過百人。

### 組織架構：「裂縫管理局」

玩家主角蘇青所屬的組織正式名稱為「裂縫管理局」（簡稱「局」），是由各國政府聯合成立的非政府軍事組織，負責：

1. 監測裂縫的分布與擴張
2. 清剿侵入陽界的異獸
3. 管理、培訓超感者
4. 研究大崩的成因與阻止方法

組織架構：

```
總部指揮官（神秘人物，未出場）
    ↓
各地區總隊長（蘭澤塔 — 火焰大隊）
    ↓
隊長（沃爾崔 — 深淵小隊）
    ↓
特攻隊員（森羅輝、夜鴉·茲凡等）
    ↓
新人/見習（蘇青、波林等）
```

局的基地位於陽界與幽界交界的「邊境都市：曉城」，一座永遠籠罩在薄霧中的城市。

### 六種屬性的世界觀解釋

超感者的能力並非隨機覺醒，而是與個人的「精神核心頻率」相關。不同的精神頻率，對應不同的能量屬性：

| 屬性 | 世界觀解釋 | 使用者特質 |
|------|-----------|-----------|
| 火（炎） | 陽界太陽能量的凝縮，代表意志與鬥志 | 熱血、衝動、領導力強的超感者 |
| 水（淵） | 幽界深處的沉靜之力，代表適應與流動 | 冷靜、多謀、善於分析的超感者 |
| 木（林） | 陽界大地生命力的結晶，代表成長與療癒 | 穩重、治癒系、與自然親近的超感者 |
| 光（曜） | 兩界交界的「裂縫之光」，代表真理與審判 | 理性、公正、感知能力超群的超感者 |
| 暗（虛） | 幽界核心的虛無能量，代表毀滅與隱匿 | 神秘、孤獨、力量最純粹也最危險的超感者 |

屬性剋制的世界觀解釋：
- 火剋木：烈焰焚盡森林
- 木剋水：根系吸乾水源
- 水剋火：洪流熄滅火焰
- 光剋暗：光明驅散虛無
- 暗剋光：黑暗吞噬光芒

    this._bgmGain = this._ctx.createGain();
    this._sfxGain = this._ctx.createGain();
    this._bgmGain.connect(this._ctx.destination);
    this._sfxGain.connect(this._ctx.destination);
    this._bgmGain.gain.value = this.bgmVolume;
    this._sfxGain.gain.value = this.sfxVolume;
  }

  async loadSfx(id, url) {
    const response = await fetch(url);
    const arrayBuffer = await response.arrayBuffer();
    this._buffers[id] = await this._ctx.decodeAudioData(arrayBuffer);
  }

  playSfx(id, pitchMultiplier = 1.0) {
    if (!this._buffers[id] || this.muted) return;
    const source = this._ctx.createBufferSource();
    source.buffer = this._buffers[id];
    source.playbackRate.value = pitchMultiplier;
    source.connect(this._sfxGain);
    source.start(0);
  }

  // COMBO 音效：音調依 COMBO 數遞增
  playComboSfx(comboCount) {
    const basePitch = 1.0;
    const pitchStep = 0.08;
    const pitch = Math.min(basePitch + (comboCount - 1) * pitchStep, 2.0);
    this.playSfx('combo_base', pitch);
  }

  // BGM 淡入淡出切換
  async switchBgm(url, fadeDuration = 500) {
    if (this._bgmNode) {
      const oldNode = this._bgmNode;
      const now = this._ctx.currentTime;
      this._bgmGain.gain.linearRampToValueAtTime(0, now + fadeDuration / 1000);
      setTimeout(() => oldNode.stop(), fadeDuration);
    }
    const response = await fetch(url);
    const arrayBuffer = await response.arrayBuffer();
    const buffer = await this._ctx.decodeAudioData(arrayBuffer);
    const source = this._ctx.createBufferSource();
    source.buffer = buffer;
    source.loop = true;
    source.connect(this._bgmGain);
    const now = this._ctx.currentTime;
    this._bgmGain.gain.setValueAtTime(0, now);
    this._bgmGain.gain.linearRampToValueAtTime(this.bgmVolume, now + fadeDuration / 1000);
    source.start(0);
    this._bgmNode = source;
  }
}
```
